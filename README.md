# VYBE Africa Website

**Live Demo**: [https://vybe-africa-website.vercel.app/index.html](https://vybe-africa-website.vercel.app/index.html)

## Overview
A modern, youth-led CBO website transforming communities in West Pokot. Redesigned with a premium "Dark Mode" aesthetic (QClay style) featuring neon accents, glassmorphism, and dynamic animations.

## Features
-   **Premium Dark Theme**: High-contrast dark mode with neon gradients.
-   **Theme Toggle**: Fully functional Light/Dark mode switcher.
-   **Responsive Design**: Mobile-first layout with custom hamburger menu.
-   **Dynamic Stats**: Animated counters for impact metrics.
-   **Interactive Gallery**: Masonry grid with filtering and lightbox access.

## Tech Stack
-   **Core**: HTML5, CSS3 (Variables), JavaScript (ES6+).
-   **Libraries**: FontAwesome (Icons), GLightbox (Media), Google Fonts (Syne/Inter).

## Deployment
Deployed on Vercel.
